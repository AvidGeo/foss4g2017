Process

We expect to have room for 200-250 talks selected through this Call for Presentations. Keynote and invited presentations will round out the progam. The volunteer FOSS4G Program Committee will lead the process but you, the attendees will help select the program through an open “Community Voting” process. The top vote getters from Community Voting will be included in the program and the Program Committee will select the remainder of the program to provide a diversity of voices and content.
